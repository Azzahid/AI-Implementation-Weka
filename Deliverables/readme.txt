=== Panduan Penggunaan ===

Penggunaan program
1. Untuk menggunakan program dengan FFNN, run file Tester.java pada direktori src/ANN/
2. Untuk menggunakan program dengan Naive Bayes, run file TesterBayes.java pada direktori src/Naive Bayes/
3. Disarankan menggunakan project di Netbeans dengan library weka.jar sudah terpasang pada project anda.
4. Happy exploring! :)
